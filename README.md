# Calculator Web App

A modern and responsive calculator built using **HTML, CSS, and JavaScript**. The application performs basic arithmetic operations with a clean dark-themed user interface and interactive button effects.

---

##  Live Demo

🔗 https://Atchut016.github.io/Calculator/

---

##  Features
-  Addition
-  Subtraction
-  Multiplication
-  Division
-  Decimal Number Support
-  Backspace Functionality
-  All Clear (AC) Button
-  Responsive Design
-  Modern Dark UI
-  Smooth Hover & Click Animations

---

## Technologies Used

- HTML
- CSS
- JavaScript 
---





##  How It Works

- HTML creates the calculator layout.
- CSS provides styling, animations, and responsive design.
- JavaScript performs calculations and handles user interactions.
